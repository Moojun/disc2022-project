Do it! C 언어 입문

C 언어를 가볍게 훑어보기 좋은 교재로, C 언어를 처음 배우는 사람에게 추천하는 교재입니다.
설명이 이해하기 쉽게 되어 있으며, 예제 문제를 통해 복습이 가능합니다.
또한 저자가 직접 찍은 강의가 youtube에 무료로 업로드되어 있으며, link 를 통해 이동할 수 있습니다.
단점은, 교재 후반부에 전처리기나 라이브러리에 대한 설명이 빈약합니다.

C 언어 코딩 도장

교재의 두께가 조금 있지만, C 언어를 처음 배우는 사람에게 추천하는 교재입니다.
교재에도 내용이 자세하게 나와 있지만, 온라인 플랫폼 을 통해
매 파트별 연습문제를 풀어볼 수 있으며, 저자의 인터넷 강의도 제공됩니다.
다만, 일부 연습문제는 난이도가 입문자 수준이 아니니,
너무 어려운 문제는 과감하게 버린 뒤 나중에 다시 보는 것을 추천합니다.

C 기초 플러스

외국에서 가장 유명한 C 언어 입문용 교재 중 하나인 'C primer plus' 의 번역본입니다.
내용이 매우 자세하며 중간중간에 있는 C 코드 예제 또한 내용 이해에 많은 도움이 됩니다.
하지만, 뒤로 갈수록 내용이 조금 어려우며, 책 페이지 수가 약 1,100 페이지로 상당히 많기 때문에
처음 C 언어를 입문하는 사람에게는 추천하지 않습니다.

강의시간이 짧으며 예제 위주의 강의로, C 언어를 빠르게 훑고 싶은 입문자에게 적합한 강의입니다.
